## What I Learned About GitHub

### Repositories
Repositories are used to store project files and version history.

### Issues
Issues help track tasks, bugs, and feature requests.

### Branches
Branches allow developers to work on changes independently.

### Pull Requests
Pull Requests are used to review and merge code changes.

### Forks
Forks are personal copies of repositories used for contribution.

### GitHub Projects
Projects help organize tasks using boards and workflows.
